API Reference

All functions are exposed via UCFTagLibrary in the Gameplay Tags Blueprint category and are available in C++. Below reflects the actual implementation in the plugin source.

Blueprint Nodes and C++ Signatures

Get Tags By Pattern

Returns tags that match a wildcard Pattern using FString::MatchesWildcard. 

// C++
UFUNCTION(BlueprintCallable, Category = "Gameplay Tags")
static TArray GetTagsByPattern(const FString& Pattern);

// Example patterns: "Combat.*", "Character.Health.*"

Does Tag Exist

Returns true if Tag.IsValid(). 

// C++
UFUNCTION(BlueprintPure, Category = "Gameplay Tags")
static bool DoesTagExist(FGameplayTag Tag);

Get Child Tags

Returns child tags of ParentTag. If bRecursive is true, returns all descendants; otherwise only direct children (one level deeper). 

// C++
UFUNCTION(BlueprintCallable, Category = "Gameplay Tags")
static TArray GetChildTags(FGameplayTag ParentTag, bool bRecursive = false);

Get Parent Tag

Returns the parent tag (segment before the last dot) or invalid if already at root. 

// C++
UFUNCTION(BlueprintPure, Category = "Gameplay Tags")
static FGameplayTag GetParentTag(FGameplayTag Tag);

Get Root Tags

Returns all top-level tags (no dots) from the registered tag set. 

// C++
UFUNCTION(BlueprintCallable, Category = "Gameplay Tags")
static TArray GetRootTags();

Get Tag Depth

Returns the depth of the tag (root = 1). Depth is the number of segments separated by dots. 

// C++
UFUNCTION(BlueprintPure, Category = "Gameplay Tags")
static int32 GetTagDepth(FGameplayTag Tag);

Is Leaf Tag

Returns true if the tag has no direct children. 

// C++
UFUNCTION(BlueprintPure, Category = "Gameplay Tags")
static bool IsLeafTag(FGameplayTag Tag);

Get All Tags

Returns a container of all tags as known by UGameplayTagsManager. 

// C++
UFUNCTION(BlueprintCallable, Category = "Gameplay Tags")
static FGameplayTagContainer GetAllTags();

Tag To Display String

Returns the last segment of the tag (after the final dot) for UI-friendly labels. 

// C++
UFUNCTION(BlueprintPure, Category = "Gameplay Tags")
static FString TagToDisplayString(FGameplayTag Tag);

Make Tag From String

Requests a tag by string. Returns invalid if it doesn't exist (does not create new tags). 

// C++
UFUNCTION(BlueprintCallable, Category = "Gameplay Tags")
static FGameplayTag MakeTagFromString(const FString& TagString);

Notes

  • Wildcard Matching: Uses MatchesWildcard. Pattern examples: Combat.*, Character.*.Low.
  • Tag Sources: Tags come from your configured Gameplay Tag tables/sources in Project Settings.
  • Performance: Prefer caching heavy queries (e.g., all descendants) during gameplay.
