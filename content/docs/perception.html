Perception

Use UCINPerceptionComponent to manage multiple senses, track stimuli, and react via events.

Setup

// Blueprint
// Add UCINPerceptionComponent to your agent actor
InitializePerception(PerceptionConfig)
  • FCINPerceptionComponentConfig: UpdateFrequency, MaxTrackedStimuli, StimulusMemoryDuration, MinDetectionConfidence, etc.

Register senses

Register a sense via interface ICINPerceptionSense and a unique ID:

// Blueprint
RegisterSense("Vision.Main", VisionSense, VisionConfig)
SetSenseEnabled("Vision.Main", true)

Queries and events

  • GetDetectedStimuli(), GetStimuliOfType(Type), GetNearestStimulus(Location), GetStimuliInRange(Location, Range)
  • Events: OnStimulusDetected, OnStimulusLost, OnPerceptionUpdated
🎯
Real-life example: When OnStimulusDetected for Enemy fires, increase an Alertness parameter via SetDirectParameter or drive a curve that decays over time.
