Getting Started

This quick guide shows how to start using the Gameplay Tags Manager utilities in Blueprints and C++.

Blueprints

  1. Open any Blueprint (e.g., an Actor or a Utility Blueprint).
  2. Right-click in the graph and search for the Gameplay Tags category.
  3. Use nodes like Get Tags By Pattern, Get Root Tags, Get Child Tags, or Tag To Display String.

Patterns

Use wildcard patterns such as Combat.* or Character.Health.* with Get Tags By Pattern.

C++

// Example usage from C++
#include "CFTagLibrary.h"

void UMySubsystem::Demo()
{
    // Get all root tags
    TArray Roots = UCFTagLibrary::GetRootTags();

    // Check if a tag exists
    FGameplayTag Health = UCFTagLibrary::MakeTagFromString(TEXT("Character.Health"));
    const bool bHasHealth = UCFTagLibrary::DoesTagExist(Health);

    // Get children (direct)
    TArray DirectChildren = UCFTagLibrary::GetChildTags(Health, /*bRecursive*/ false);

    // Get children (recursive)
    TArray AllDescendants = UCFTagLibrary::GetChildTags(Health, /*bRecursive*/ true);

    // Display names
    for (const FGameplayTag& Tag : AllDescendants)
    {
        UE_LOG(LogTemp, Log, TEXT("%s (Depth %d)"), *UCFTagLibrary::TagToDisplayString(Tag), UCFTagLibrary::GetTagDepth(Tag));
    }
}
