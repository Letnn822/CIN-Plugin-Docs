Contributing to CIN Plugin

Guidelines for contributing code, documentation, examples, and feedback to the CIN Plugin project.

10 minutes Guide

Ways to Contribute

1. Code Contributions

  • Bug fixes
  • New features
  • Performance improvements
  • New archetypes or services
  • Test coverage

2. Documentation

  • Fix typos or errors
  • Add examples
  • Improve clarity
  • Translate to other languages
  • Create video tutorials

3. Community Support

  • Answer questions in forums
  • Help other users
  • Share your projects
  • Write blog posts

4. Testing & Feedback

  • Report bugs
  • Test new features
  • Suggest improvements
  • Share use cases

Getting Started

Development Setup

1. Fork the repository
2. Clone your fork
3. Create feature branch
4. Make changes
5. Test thoroughly
6. Submit pull request

Code Standards

  • Follow Unreal Engine coding standards
  • Add comprehensive comments
  • Include unit tests for new features
  • Update documentation
  • Maintain backward compatibility

Pull Request Process

  1. Create Issue - Discuss feature/fix first
  2. Fork & Branch - Work in feature branch
  3. Implement - Make your changes
  4. Test - Ensure nothing breaks
  5. Document - Update relevant docs
  6. Submit PR - Detailed description
  7. Code Review - Address feedback
  8. Merge - Once approved

Bug Reports

Good Bug Report Includes

  • Clear description of issue
  • Steps to reproduce
  • Expected vs actual behavior
  • Engine version
  • Plugin version
  • Relevant code/configuration
  • Screenshots or logs

Feature Requests

Effective Feature Request

  • Describe the problem it solves
  • Explain use case
  • Provide examples
  • Consider alternatives
  • Discuss implementation ideas

Code of Conduct

  • Be respectful and professional
  • Welcome newcomers
  • Accept constructive criticism
  • Focus on what's best for the community
  • Show empathy

Recognition

All contributors are acknowledged in release notes and documentation. Significant contributions may result in maintainer status.

Questions?

Open a discussion topic or reach out through official channels.
