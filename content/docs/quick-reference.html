Quick Reference Guide

Fast lookup for common tasks, functions, and configurations. Perfect for experienced users who need quick answers.

10 minutes Reference

Quick Setup (5 Minutes)

1. Add to AIController/Pawn: CINQuickSetupComponent
2. Set AgentType: BaseCombat (or desired archetype)
3. Enable bAutoInitialize: true
4. Add AIPerceptionComponent (if needed)
5. Set Team ID: SetGenericTeamId(1)
6. Done!

Action Configuration Cheat Sheet

BaseScore:
  30-40: Low priority / Fallback
  50-60: Normal actions
  70-80: Important actions
  85-95: Critical/survival actions

Priority:
  0-30: Low
  40-60: Normal
  70-90: High

ReplaceResistance:
  1.0: Easy to interrupt
  1.5: Moderate resistance
  2.0+: Hard to interrupt

Momentum:
  MaxMomentum: 1.0-3.0
  MomentumDecayTime: 2.0-5.0s
  
UnlockTime:
  0.0: No cooldown
  2.0-5.0: Short cooldown
  10.0+: Long cooldown
  -1.0: Locks permanently

Consideration Patterns

// Linear (proportional)
(0,0) → (1,100)

// Boolean (required)
(0,0) → (0.99,0) → (1,100)

// Threshold (activate at value)
(0,0) → (0.5,0) → (0.5,100) → (1,100)

// Inverse (prefer low values)
(0,100) → (1,0)

// Bell curve (optimal range)
(0,0) → (0.5,100) → (1,0)

// Exponential (urgency)
(0,0) → (0.7,30) → (0.9,70) → (1,100)

Service Configuration

MinInterval:
  0.05s: Very fast (use sparingly)
  0.1s: Fast (perception, health)
  0.2s: Normal (most services)
  0.5s: Slow (inventory, quests)
  1.0s+: Very slow (strategic analysis)

Common Fixes

AI not doing anything:
✓ Check brain has actions
✓ Verify services publishing inputs
✓ Ensure considerations can score > 0

AI switching too much:
✓ Increase ReplaceResistance
✓ Add Momentum
✓ Smooth consideration curves

AI too slow to respond:
✓ Decrease TickInterval
✓ Increase action BaseScore
✓ Check ReplaceResistance not too high

Performance issues:
✓ Reduce action/consideration count
✓ Increase service MinInterval
✓ Implement LOD system
✓ Use stat ai to profile
