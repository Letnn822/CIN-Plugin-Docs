Installation

These steps assume an Unreal Engine 5.6+ project on Windows.

1) Verify plugin

  • Plugin is located at Plugins/CINPlugin/ in your project.
  • Open your project. In Edit → Plugins, ensure CINPlugin is enabled. Modules include CINCore and (optionally) CINPerception.

2) Build

  • Regenerate project files if needed and build the project (Development Editor / Win64).
  • Confirm the editor loads without errors.

3) Quick verification

  • Create a Blueprint derived from ACINAgentBase.
  • Place it in a level. Press Play. The agent will tick and maintain a context even before you bind parameters.
Optional: Word-based DOCX/PDF export scripts are in Intermediate/DocGen/ (ExportDocx.ps1, DocxToPdf.ps1). These are not required for runtime.
