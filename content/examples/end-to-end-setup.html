End-to-End Setup

An example from empty level to behavior switching.

1) Agent + Health

// Blueprint
BindToGameplayAttribute("Health", "Attributes.Health", true, (0, 100))

2) Stamina + Distance

// Blueprint
BindToGameplayAttribute("Stamina", "Attributes.Stamina", true, (0, 100))
BindToBlueprintFunction("DistanceToPlayer", "GetDistanceToPlayerNormalized")

3) Behavior switch

// Pseudo Blueprint scoring
ScoreAdvance = (1 - DistanceToPlayer) * Stamina
ScoreRetreat  = (1 - Health)
SetCurrentGoal(Max(Advance, Retreat))

4) Validate

ValidateParameterBindings()
RefreshParameterBindings()
