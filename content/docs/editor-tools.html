Editor Automation: Create Demo AI Trees + Blackboard

This editor tool generates a demo-ready Blackboard and Behavior Trees for quick iteration and testing. It is fully implemented in C++ and requires no manual setup after running.

Menu Path

Tools → CIN → Create Demo AI Trees + Blackboard

What it creates

  • Folder: /Game/CIN/Demo/AI
  • Blackboard: BB_Soldier
  • Behavior Trees: BT_Soldier_Search, BT_Soldier_FlankLeft, BT_Soldier_FlankRight

Blackboard keys ensured

  • TargetLocation (Vector)
  • HasKnownEnemy (Bool)
  • StrategyName (Name)
  • IsLeader (Bool)
  • RoleIndexNormalized (Float)
  • KnownEnemiesCount (Int)
  • LastKnownEnemyDistance (Float)

Behavior Tree scaffolding

Each created BT is assigned to BB_Soldier and gets a root Sequence with the UBTService_SquadBlackboard service attached. The service key selectors are set to the Blackboard keys listed above.

Default nodes

  • Search: MoveTo(TargetLocation, 250) → Wait(1.5s)
    • Blackboard decorator on HasKnownEnemy: Not Set
  • FlankLeft / FlankRight: MoveTo(TargetLocation, 200)
    • Blackboard decorator on HasKnownEnemy: Set
Idempotent behavior: Scaffolding happens only if a BT has no children. If you hand-edit a BT later, re-running the tool will not duplicate nodes.

Strategy profile wiring

The tool updates your strategy profile CIN_Demo_Strategy with mappings:

  • Search → BT_Soldier_Search
  • FlankLeft → BT_Soldier_FlankLeft
  • FlankRight → BT_Soldier_FlankRight
  • Assault → BT_Soldier_Assault (only if that BT already exists)

It also stores the common Blackboard key names into the profile's BlackboardKeys struct for convenience.

How to validate

  1. Open /Game/CIN/Demo/AI/BB_Soldier and verify the keys listed above.
  2. Open each BT and confirm the root Sequence, service, and default nodes were added as described.
  3. Open /Game/CIN/Demo/CIN_Demo_Strategy and verify strategy mappings and Blackboard key names.

Troubleshooting

  • No assets created? Ensure the project builds and the editor module has dependencies: AIModule, BehaviorTreeEditor.
  • Want different defaults? You can change acceptable radius or wait time later by editing the BTs manually.
