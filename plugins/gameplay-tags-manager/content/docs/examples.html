Examples

Practical snippets showing how to use Gameplay Tags Manager features in Blueprints and C++.

Find Tags by Wildcard Pattern

Get all tags under a category like Combat.*.

// C++
TArray CombatTags = UCFTagLibrary::GetTagsByPattern(TEXT("Combat.*"));
for (const FGameplayTag& Tag : CombatTags)
{
    UE_LOG(LogTemp, Log, TEXT("%s"), *Tag.ToString());
}

List Root Tags

// C++
TArray Roots = UCFTagLibrary::GetRootTags();

Navigate Hierarchy

// C++
FGameplayTag Health = UCFTagLibrary::MakeTagFromString(TEXT("Character.Health"));
FGameplayTag Parent = UCFTagLibrary::GetParentTag(Health);
TArray DirectChildren = UCFTagLibrary::GetChildTags(Health, /*bRecursive*/ false);
TArray AllDescendants = UCFTagLibrary::GetChildTags(Health, /*bRecursive*/ true);

Blueprint Quick Tips

  • Use Get Tags By Pattern to build dynamic menus (e.g., list all Character.* tags).
  • Use Tag To Display String to show a friendly label in UI widgets.
  • Check Does Tag Exist before operating on user-input strings.
⚠️
Large Tag Sets

Cache frequently used results (e.g., descendants) in your subsystems to avoid repeating expensive queries.
