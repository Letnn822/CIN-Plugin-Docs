Beginner Learning Path

Build a working agent in minutes.

Step 1 — Create an Agent Blueprint

  1. Create a BP derived from ACINAgentBase. Name it BP_CINAgent.
  2. Place it in a test level.

Step 2 — Bind a first parameter

// Blueprint (BeginPlay)
SetDirectParameter("Curiosity", 0.5)

Confirm in a debug print that GetAgentParameter("Curiosity") returns 0.5.

Step 3 — Bind Health

// With a Health component (0..100)
BindToObjectProperty("Health", HealthComponent, "CurrentHealth", true, (0, 100))

Step 4 — Use parameters in logic

// Pseudo Blueprint
If (Health < 0.25) -> SetCurrentGoal("Retreat")
Else -> SetCurrentGoal("Advance")
Keep iterating with GAS attributes and perception once the basics work.
