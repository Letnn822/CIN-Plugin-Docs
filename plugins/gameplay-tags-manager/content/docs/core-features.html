Core Features

Gameplay Tags Manager focuses on convenient, Blueprint-friendly utilities built on Unreal's native Gameplay Tags framework.

Hierarchy Utilities

  • Get Root Tags: Return all top-level tags (no dots).
  • Get Child Tags: Direct or recursive descendants of a given parent tag.
  • Get Parent Tag: Returns the parent (segment before the last dot) or invalid if already root.
  • Get Tag Depth: Returns the hierarchy depth (root = 1).
  • Is Leaf Tag: True if the tag has no direct children.

Runtime Query Utilities

  • Get Tags By Pattern: Wildcard search using MatchesWildcard (e.g., Combat.*).
  • Get All Tags: Returns a FGameplayTagContainer of all registered tags.
  • Does Tag Exist: Validity check using FGameplayTag::IsValid().
  • Make Tag From String: Requests an existing tag; returns invalid if it doesn't exist.

Display Helpers

  • Tag To Display String: Returns the last segment (after final dot) for UI-friendly labels.
⚙️
Performance

Tag queries leverage UGameplayTagsManager::RequestAllGameplayTags and simple string checks. For large tag sets, prefer caching results you reuse across frames.
