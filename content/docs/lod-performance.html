LOD & Performance

Scale AI and animation cost based on agent importance and distance.

Key hooks on ACINAgentBase

  • GetIntelligenceLOD(), SetIntelligenceLOD(NewLOD)
  • Event: OnLODChanged
  • Settings: ContextUpdateFrequency, AgentImportanceRadius, bEnableLODScaling, bEnableDebugVisualization

Patterns

  • Throttle expensive queries (perception scans, line traces) when LOD is low.
  • Reduce AnimBP complexity and IK at distance or low importance.
  • Skip non-critical updates when off-screen and far from the player.

Example

// Blueprint
OnLODChanged -> Switch on ECINIntelligenceLOD
Low:    Disable complex perception
Medium: Enable minimal set
High:   Enable full feature set
