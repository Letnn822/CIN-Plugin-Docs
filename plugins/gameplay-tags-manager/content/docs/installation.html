Installation

This plugin targets Unreal Engine 5.5–5.6. You can install it to your project as a standard Engine/Project plugin.

Requirements

  • UE 5.5 or 5.6
  • Gameplay Tags enabled (Project Settings → Gameplay Tags)

Steps

  1. Copy the plugin into your project at Plugins/CFTagManager.
  2. Enable the plugin in Edit → Plugins then restart the editor if prompted.
  3. Configure Gameplay Tags in Project Settings → Gameplay Tags. Add or import your tag list and set tag sources as needed.
  4. Verify by opening any Blueprint and searching for nodes in the Gameplay Tags category like Get Tags By Pattern or Get Root Tags.
✅
Packaging

Nothing special is required for packaging. The plugin relies on UE's built-in Gameplay Tags system.
