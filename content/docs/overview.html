Smart Action Selection

Agents score actions dynamically and pick the best choice every frame. No rigid trees, just intelligent adaptation.

  • Task Momentum - Prevents action spam
  • Replace Resistance - Reduces thrashing
  • Auto Locking - Failed actions lock temporarily
  • Visual Curves - Easy tuning without code
// Actions score themselves
Attack:  HasEnemy × (1 - Distance) × Ammo
Reload:  (1 - Ammo) × 2.0
Cover:   ThreatLevel × (1 - Health)
Retreat: (1 - Health) × Threat × Alone

// Brain picks highest score
// Momentum prevents spam
